This is a profile script for Windows Powershell. It contains customize
settings. This profile is run everytime a new session of Windows Powershell is
started.

Note for Powershell to beable to run this profile scripts, make sure the
Execution Policy is set to at least RemoteSigned. To view the ExecutionPolicy,
run the following command.

>Get-ExecutionPolicy

To set the Execution Polciy to RemoteSigned, execute the following command

>Set-ExecutionPolicy RemoteSigned
